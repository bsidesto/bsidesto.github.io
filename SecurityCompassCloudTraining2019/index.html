Security Compass Day One Training: Basic Cloud Security - Securing AWS

About this Event

Security Compass will be running a one day training session on Saturday October 5th titled Basic Cloud Security - Securing AWS

This one-day training session in designed as an introduction to AWS Security fundamentals. It is targeted to those with little or no experience with using AWS for production workloads, who would like to quickly understand security basics and best practices. It will also include guidance on leveraging this information within an organisation to begin building a Cloud Security Programme. This course will include a hands-on component, where students will be allowed to leverage the techniques being discussed.

Areas to be Covered

  • Introduction to AWS Security
  • Access Management (IAM and Security Policies)
  • Logging and Monitoring
  • Infrastructure Security
  • Data Protection
  • AWS Security in the Real World

Requirements
In order to take part in the hands-on exercises, students will require the following:

  • Laptop with AWS CLI installed (https://docs.aws.amazon.com/cli/latest/userguide/cli-chap-install.html)
  • Validated AWS Account (with root user access)

25 students maximum, those that arrive unprepared will be replaced.
